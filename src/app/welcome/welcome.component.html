Who are we?

DefineX Creative Studio developes API, softwares and applications to ease the work and they also develop small games for entertainment of others. Most of the projects build by DCS are open-source.

VJTI kit is an open source project with MIT license, so you can use its source code however you desire. Help us adding more useful features for the students of VJTI yet to come, or just create an issue if you encounter any bugs or have a feature request.
